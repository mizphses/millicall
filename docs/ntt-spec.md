# NTT関連の制約について

本プロジェクトはNTTの「スマホ de ひかり電話」という機能に強く依存しています。そのため、一部非対応気があったり、若干面倒な設定を行う必要があります。ちなみにこの辺の設定ノウハウはそのままAsterisk収容にも使えます。

## 対応機かの確認

あまりに古い端末機だと非対応のケースがあります。

> HGW、ONUなど言葉の意味がわからない人は、以下の手順で使えるかどうかを確認してください。なお、大前提として固定電話の契約があるものとします。
> 
> 確認方法A
> * WiFiに繋がっている端末でHGWのアドレス(大抵は[192.168.1.1](http://192.168.1.1))を開き、電話設定→内線設定の画面で「IPphone1」などIP電話が設定できそうな画面があることを確認する。
> 
> 確認方法B
> * まず、NTTの機械(SoftbankやKDDIではないですよ!)が2台来ている場合は、電話番号の契約があれば大体大丈夫です。
> * NTTからの機械が1台しか来ていない場合、それに白いケーブルが刺さっており、かつ同じ機械にLANポートが4つくらいついている機械であれば問題ありません。
> * [こちらのサイト](https://flets.com/device/hgw/)でその機械の品番を探します。個別ページに「一体型のひかり電話ルーターです。本商品は、ひかり電話対応機器です。」または「単体型のホームゲートウェイです。」との記載があれば大体ご利用になれます。

ひかり電話の直収については、この機能が有効なフレッツ光回線のみがサポート対象であり、以下のようなケースについてはサポートされていません。

* そもそもフレッツ回線ではない場合（Nuro、auひかり、コミュファ光、eo光など）←NTTの機械が来ているかどうかで判断ください。
* 家庭用のひかり電話を契約していない場合
  * フレッツ光だが、ひかり電話ではない電話回線の場合もあります。（特にSoftbank光については光電話(N)・BBフォンなどすべて対象外です。）
  * ひかり電話オフィスタイプなど、家庭用のフレッツ光回線ではない場合（検証環境がないためサポートできません）
  * ひかり電話の契約がないが、WiFiルーターレンタルなどのオプションによってホームゲートウェイがある可能性があります。
* ひかり電話をホームゲートウェイで収容していない場合
  * NTTから黒い大きい箱型の機械(HGW)が必ずあり、場合によってはさらに背の低い黒または白の箱型の機械(ONU)が来ています。"背の低い１個"（光ファイバーとLANポートが1個ずつのもの）しかない場合は非対応のケースが多いです。

## 設定方法

XG-100NEしか持っていないので、他機種については誰か実験させてくれるか追記してくれたらマニュアルにします。

1. IPPhoneのどれかについている「編集」ボタンを押します。
2. 端末属性を「音声専用端末」にします。
3. ダイジェスト認証を「行う」に設定します。
4. 内線番号とユーザー名を同じ数字にします。例えば、内線番号40→ユーザIDも40です。
5. パスワードは好きなもので結構です。
